📦 REQUISITOS:
═══════════════════════════

Necesitas instalar ANTES de compilar:

☐ Java JDK 11+ (https://www.oracle.com/java/)
☐ Android SDK (https://developer.android.com/studio)
☐ Configurar variables de entorno (JAVA_HOME, ANDROID_HOME)

Puedes usar check-requirements.ps1 para verificar si está todo.

💻 SCRIPTS NPM DISPONIBLES:
═══════════════════════════

npm run build              - Compila el proyecto
npm run cap:sync          - Sincroniza cambios con Android
npm run cap:open          - Abre el proyecto en Android Studio
npm run apk:debug         - Compila APK de debug (todo en uno)
npm run apk:release       - Compila APK para producción
